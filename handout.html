Getting started quickly with Claude Code

Follow the link above or follow the steps below:

1
Install
Open Terminal, paste the command, and press Enter.
curl -fsSL https://claude.ai/install.sh | bash
Read the output carefully. If asked, copy/paste the command shown and press Enter.
2
Start Claude
Type "claude", hit Enter, and finish signing in.
claude
Select "Claude.ai" when asked how to authenticate.
3
Build
Describe what you want to build and press Enter.
Enter planning mode: Generate a Product Requirements Document and help me build...
Replace "..." with your project idea.
